    문제#

    Intellij 내에서 간단한 문구 수정 때문에 properties file을 수정했는데 갑자기 file 전체에 변경사항이 잡혔다! 내용을 확인해보니 utf-8 Code 의 제일 앞 알파벳이 다 대문자로 upper 된 것 이였다..;

    원인#

    사실 크게 문제가 될 것 같지는 않지만 코드 리뷰 페이지에서 file 전체에 변경사항이 잡혀 edit line이 쓸데없이 많이 counting 되었다. 또한 호오오오옥~~ 시나 모르는 bug가 발생할 가능성이 무서워.. 수정을 해야할 것 같았다!

    문제의 원인은 project를 처음에 import 할 때에 eclipse type으로 열면 project option이 auto upper로 설정되는 듯 하였다! (maven, gralde로 import한 프로젝트에서는 발생하지 않았다!)

    해결 방법#

    문제 해결방식은 단순했다! intellij global option을 설정하는 file (아마도) intellij > bin > idea.properties file 내에 idea.native2ascii.lowercase=true 만 추가해주면 됐다!

    사소한 문제지만 기억해두면 찾을 필요가 없을 것 같아 posting 해둔다 ㅎㅎ..
